<input type="text">

É um atributo usado para criar um campo onde o usuário pode digitar texto. Isso é comum em formulários, onde precisamos de informações como nome ou endereço por exemplo.

X

<input type="email">

É um atributo usado para criar um campo de entrada onde o usuário deve digitar um endereço de email. Esse tipo de campo oferece uma validação básica, garantindo que o texto inserido tenha o formato de um email (como usuario@exemplo.com).

X

<input type="password">

É um atributo usado para criar um campo de entrada onde o usuário pode digitar uma senha. O texto digitado é mascarado, aparecendo como pontos ou asteriscos, para manter a privacidade da informação.

X

<input type="checkbox">

É um atributo usado para criar uma caixa de seleção, permitindo que o usuário escolha uma ou mais opções de uma lista. O usuário pode marcar ou desmarcar a caixa conforme sua escolha.

X

<input type="date">

É um atributo usado para criar um campo de entrada que permite ao usuário selecionar uma data através de um calendário interativo. O formato da data é padronizado, garantindo uma entrada correta.

X

<input type="color">

É um atributo usado para criar um campo de entrada que permite ao usuário selecionar uma cor a partir de um seletor de cores. O valor inserido é um código de cor em hexadecimal.

X

<input type="button">

É um atributo usado para criar um botão que pode ser clicado, mas que não envia dados do formulário por padrão. Pode ser utilizado para executar scripts ou ações específicas quando o botão é clicado.

X

<input type="range">

É um atributo usado para criar um campo de entrada que permite ao usuário selecionar um valor dentro de um intervalo usando um controle deslizante. O valor é representado por uma barra que o usuário pode arrastar.

X

<input type="radio">

É um atributo usado para criar um campo de entrada que permite ao usuário selecionar uma única opção de um grupo de opções. Os botões de rádio são exibidos como pequenos círculos que podem ser marcados ou desmarcados, mas apenas uma opção pode ser selecionada por vez dentro de um mesmo grupo.

X

<input type="url">

É um atributo usado para criar um campo de entrada que permite ao usuário digitar um URL. O campo valida automaticamente se o texto inserido tem o formato de um endereço de site, como https://www.exemplo.com.

X

<input type="reset">

É um atributo usado para criar um botão que, quando clicado, redefine todos os campos de um formulário para seus valores padrão ou iniciais. É útil para limpar rapidamente os dados inseridos pelo usuário.

X

<input type="datetime-local">

É um atributo usado para criar um campo de entrada que permite ao usuário selecionar uma data e uma hora em um único controle. O valor inserido inclui a data e a hora no formato YYYY-MM-DDTHH:MM.

X

<input type="month">

É um atributo usado para criar um campo de entrada que permite ao usuário selecionar um mês e um ano, sem especificar o dia. O valor inserido é no formato YYYY-MM.

X

<input type="week">

É um atributo usado para criar um campo de entrada que permite ao usuário selecionar uma semana e um ano. O valor inserido é no formato YYYY-Www, onde YYYY representa o ano e Www representa a semana do ano.

X

<input type="tel">

É um atributo usado para criar um campo de entrada que permite ao usuário inserir um número de telefone. Embora o campo não valide automaticamente o formato do telefone, ele pode ser usado para facilitar a entrada de números de telefone em formulários.

X

<input type="file">

É um atributo usado para criar um campo de entrada que permite ao usuário selecionar e enviar arquivos do seu dispositivo. O usuário pode escolher um ou mais arquivos para upload, dependendo das configurações.

X

<input type="hidden">

É um atributo usado para criar um campo de entrada que não é exibido ao usuário, mas pode armazenar informações que são enviadas junto com o formulário. É útil para passar dados que não precisam ser alterados pelo usuário, mas são necessários para o processamento do formulário.

<input type="image">

É um atributo usado para criar um botão de envio de formulário que é exibido como uma imagem. Quando o usuário clica na imagem, o formulário é enviado. O atributo type="image" permite que a imagem funcione como um botão de envio, e os valores das coordenadas do clique podem ser enviados junto com o formulário.

X

<input type="search">

É um atributo usado para criar um campo de entrada que permite ao usuário digitar termos de pesquisa. Esse campo é semelhante ao type="text", mas é otimizado para inserção de termos de busca, podendo exibir funcionalidades específicas, como botões de limpar texto.

X

<input type="number">

É um atributo usado para criar um campo de entrada que permite ao usuário digitar apenas números. O campo pode incluir setas para aumentar ou diminuir o valor, e você pode definir valores mínimos, máximos e incrementos.

X
